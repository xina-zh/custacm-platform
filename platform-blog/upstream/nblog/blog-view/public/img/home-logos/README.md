# 首页标志资源

本目录保存首页首图下方展示的赛事与校队标志。

| 文件 | 责任与来源 |
| --- | --- |
| `icpc-foundation.svg` | ICPC 官方 Press 页面提供的透明 SVG 标志，来源：<https://news.icpc.global/press/> |
| `ccpc.png` | CCPC 官网使用的透明 PNG 标志，来源：<https://ccpc.io/uploads/20220815/ccpc_logo.png> |
| `custacm-round.png` | 用户提供的 CUSTACM JPG 标志；保留完整外圈、文字和装饰，只将方形画布裁成透明圆形并缩放为 640×640 |

这些文件只用于首页静态展示；替换资源时需保持透明背景，并同步更新本表与 `blog-view/README.md`。
