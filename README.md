# custacm-platform

`custacm-platform` 是 CUST XCPC 集训队的日常工作台。

本项目提供队员身份管理、OJ 账号绑定、训练数据采集、训练记录查询和管理员维护能力。队员登录后，可以查看自己的训练记录、提交明细和通过题目；管理员可以维护队员信息和 OJ 账号，发起数据采集，也可以在需要时清理训练数据。

后续还会支持云文档能力，用来宣传队伍、分享训练心得、编写题解和沉淀训练经验，为新老队员交流提供平台。
