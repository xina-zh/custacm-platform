# custacm-platform

`custacm-platform` 是 CUST ACM 集训队的日常工作台，包含队员与 OJ 账号管理、训练数据采集和查询，以及用于沉淀题解、复盘和队内动态的 Blog。

平台欢迎访客浏览公开文章与赛事记录；登录队员可以查看训练数据、维护个人资料并发布内容，管理员负责账号、文章、赛事和采集运维。
