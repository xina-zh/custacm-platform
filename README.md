# custacm-platform

`custacm-platform` 是 CUST XCPC 集训队的日常工作台。

本项目提供队员身份管理、OJ 账号绑定、训练数据采集、训练记录查询和管理员维护能力。队员登录后，可以查看自己的训练记录、提交明细和通过题目；管理员可以维护队员信息和 OJ 账号，发起数据采集，也可以在需要时清理训练数据。

同时也提供了一个博客平台。我们希望队员们能在这里沉淀训练经验、题解笔记、技术资料与队内动态，也为大家提供一个方便讨论、互相学习和共同进步的空间。无论是比赛复盘、算法分享、学习心得，还是日常训练中遇到的问题，都欢迎在这里记录和交流哦哦。

当然，我们也欢迎各位访客前来参观我们的平台，了解集训队内的训练日常与成长历程。希望这里的内容能为你带来一些帮助与启发，也期待与更多热爱算法和程序设计的朋友相遇。
